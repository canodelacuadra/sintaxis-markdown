# Sintaxis de Markdown
Este es un documento que recogerá la sintaxis básica de markdown
- Encabezados
- Negritas y cursivas
- Listas ordenadas y desordenadas
- Inserción de código fuente
- Enlaces
- Imágenes
- Líneas horizontales
- Tablas
- Comentarios (no sé)
- citas
